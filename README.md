# Factory Downtime Analysis (Deloitte Virtual Internship)

This project was created during the Deloitte Virtual Internship program.  
It analyzes factory downtime data using Tableau dashboards.

## Tools Used
- Tableau Public
- Microsoft Excel

## Insights
- Identified factories with highest downtime
- Compared device types by performance
- Visualized key production issues

## Output
The dashboard shows downtime per factory and device type to improve efficiency.
